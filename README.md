# ✨ Savoria Digital Jar

**Savoria Digital Jar** adalah aplikasi web interaktif yang berfungsi sebagai "Toples Syukur Digital". Di tengah sibuknya kehidupan, aplikasi ini didesain sebagai tempat pelarian yang nyaman dan estetik untuk mencatat hal-hal kecil yang kamu syukuri setiap hari. 

Kapan pun kamu merasa sedih atau butuh penyemangat, kamu bisa membuka toples ini secara acak untuk membaca kembali kepingan memori bahagia di masa lalu.

---

## 🌟 Fitur Utama
- **UI/UX Premium (Glassmorphism):** Didesain dengan sangat elegan menggunakan perpaduan efek kaca tembus pandang, warna pastel yang hangat, dan tipografi klasik bergaya jurnal.
- **Toples Interaktif:** Animasi visual 3D saat menyimpan catatan, dan animasi kertas berterbangan saat membuka memori secara acak.
- **Memori Tanpa Batas:** Tuliskan rasa syukurmu tanpa dibatasi kuota harian. Seluruh catatan di hari yang sama akan dirangkum rapi ke dalam satu kartu elegan.
